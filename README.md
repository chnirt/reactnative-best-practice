### Navigator structure

    AppNavigator                        # Stack
    ├── SplashScreen
    ├── OnBoardingScreen
    ├── AuthStackNavigator              # Stack
    │   ├── LoginScreen
    │   ├── RegisterScreen
    │   └── ForgotScreen
    └── DrawerNavigator                 # Drawer
        └── DashboardNavigator          # Stack
            ├── TabNavigator            # BottomTab
            │   ├── HomeScreen
            │   ├── MessageScreen
            │   ├── NotificationScreen
            │   └── ProfileScreen
            └── PostScreen
